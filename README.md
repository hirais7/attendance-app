# 出退勤打刻アプリ 研修課題

スマートフォンからワンタップで出退勤を記録し、LINEグループに通知するアプリです。

## 🚀 セットアップ手順

### 1. Google スプレッドシートの作成
1. 指定されたスプレッドシート（ID: `19_2PnWEyRxKqvg8k1uDhBjC9TdRe2R1aLi8o1lZXa8`）を使用します。
2. 以下の3つのシートが作成されていることを確認してください：
   - **研修生マスタ**: `研修生ID`, `氏名`, `ステータス`
   - **打刻記録**: `日付`, `研修生ID`, `氏名`, `出勤時刻`, `退勤時刻`, `勤務時間`
   - **課題完了記録**: `完了日時`, `研修生ID`, `氏名`, `アプリURL`, `判定`
3. 「研修生マスタ」に自分自身の情報を入力しておきます。
   - `user01`, `あなたの名前`, `進行中`

### 2. Google Apps Script (GAS) のデプロイ
1. スプレッドシートの「拡張機能」 > 「Apps Script」を開きます。
2. `gas/code.gs` の内容をコピー＆ペーストします。
3. 右上の「デプロイ」 > 「新しいデプロイ」を選択します。
4. 種類を「ウェブアプリ」に設定します。
5. アクセスできるユーザーを「全員」に設定してデプロイします。
6. 発行された **ウェブアプリのURL** をコピーします。

### 3. フロントエンドの設定
1. `script.js` を開き、1行目の `GAS_URL` にコピーしたURLを貼り付けます。
   ```javascript
   const GAS_URL = 'https://script.google.com/macros/s/.../exec';
   ```

### 4. GitHub Pagesへのデプロイ
1. このリポジトリをGitHubにプッシュします。
2. リポジトリの Settings > Pages から「Deploy from a branch」を選択し、`main` ブランチを公開します。
3. 公開されたURLがアプリのURLになります。

### 5. PWAとしてインストール
1. スマホのブラウザ（iOSならSafari, AndroidならChrome）でアプリURLを開きます。
2. 「ホーム画面に追加」を選択すると、アプリアイコンから起動できるようになります。

## 🎯 使い方
1. 出勤時に「出勤」ボタンを押します。LINEに通知が飛びます。
2. 退勤時に「退勤」ボタンを押します。勤務時間が計算され、LINEに通知が飛びます。
3. 全ての動作確認ができたら「🎉課題完了報告」ボタンを押して提出してください。

## 🔧 技術情報
- **フロントエンド**: HTML5, CSS3 (Modern Glassmorphism), JavaScript (Vanilla)
- **バックエンド**: Google Apps Script
- **通知**: LINE Messaging API
- **PWA**: manifest.json, Service Worker対応
